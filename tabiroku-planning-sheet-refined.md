# 旅録 事業計画シート 改訂版

元の表をそのまま使うには少し大きすぎたので、実際に進めやすいように整理しました。

## 直したポイント

- 重複していた論点をまとめました
- `目的 / 成果物 / 開発に必要な情報 / 進め方` の役割が分かるように文量をそろえました
- 同じ粒度になるように、1行 = 1つの検討テーマに寄せました
- MVP前提で優先順位がつけやすい順番に整えました
- そのままスプレッドシートに戻しやすいように CSV で保存しました

## ファイル

## まず先にやるべき10項目

1. `目的の確認 Why`
2. `成功条件の確認`
3. `サービス定義 What`
4. `主要顧客の確定 Who`
5. `送り主ニーズ整理`
6. `旅行者ニーズ整理`
7. `収益ポイント整理`
8. `パイロットモデル設定 How`
9. `パイロット地域条件設定 Where`
10. `最小機能 MVP整理`

## この改訂版の使い方

- まずは上の10項目だけ埋める
- 次に `体験設計` と `アプリ要件定義` へ進む
- `営業準備` と `運用設計` は MVP の輪郭が見えてから詳細化する

## 次におすすめの整理

- `MVPでやらないこと` の列を追加する
- `優先度` と `担当者` の列を追加する
- `意思決定日` と `決定者` の列を追加する

必要なら次は、この改訂版をベースに `優先度付きの実行ロードマップ版` まで作れます。
